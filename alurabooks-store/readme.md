<div class="about">
    <p>
        Esse projeto do AluraBooks mostra uma página com diferentes livros que foram adicionados através de uma API. Foram criados arquivos JS para dinamizar a tela, de forma que mostre os livros de acordo com suas categorias. Na categoria de livros disponíveis há o calculo automatizado de todos estes livros e em todas as telas os livros indisponíveis ficam opacos.
